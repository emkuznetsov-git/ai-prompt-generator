# Генератор промптов для AI - Fashion/Beauty/Lingerie

Генератор промптов для создания изображений в стиле fashion/beauty/lingerie съёмок.

## Структура проекта

- `config.json` - конфигурация со всеми модулями и вариантами
- `index.html` - главная страница с интерфейсом
- `style.css` - стили интерфейса
- `script.js` - логика генерации промптов
- `Description.md` - описание задачи и требований

## Запуск

1. Откройте `index.html` в браузере
2. Выберите параметры в интерфейсе
3. Нажмите "Сгенерировать промпт"
4. Скопируйте результат

## Особенности

- Модульная структура конфигурации - легко добавлять новые варианты
- Ползунок Boldness для контроля уровня смелости контента
- Пресеты эстетики и архетипы моделей для быстрого выбора
- Современный адаптивный интерфейс

## Расширение конфигурации

Все варианты хранятся в `config.json`. Для добавления новых опций просто отредактируйте соответствующие массивы в JSON файле.
