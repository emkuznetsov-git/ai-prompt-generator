Отвечу как всемирно известный **арт-директор рекламной fashion/beauty-съёмки** (lingerie & glamour), лауреат **D&AD Yellow Pencil**.

**TL;DR**: делай генератор не “списком фетиш-атрибутов”, а как **бриф для съёмки**: _цель/платформа → арт-дирекшн → модель (adult 25+) → beauty → lingerie/shoes → сцена → поза/действие → свет → камера/композиция → грейдинг/ретушь → ограничения/negative → параметры движка_. И отдельно введи ползунок **Boldness** (смелость), чтобы не ломать стиль и модерацию.

---

## Важная оговорка про “a-la Madison Ivy / Kortney Kane”

Я не могу помогать собирать промпты, которые **сексуализируют конкретных публичных персон** или просят “сделай как X (реальный человек)” — даже если это adult-индустрия.  
Но могу дать **набор “архетипов”** (типаж + вайб + стайлинг) и **пресеты эстетики Playboy/glamour**, чтобы ты получал тот же _жанровый_ результат без привязки к конкретным людям.

## 1) Твой конструктор — расширенная структура модулей

### A) Движок и выходной формат

- **Engine**: Midjourney / SDXL / Flux / DALL·E и т.д.
    
- **Формат**: Instagram feed (4:5), story/reels (9:16), журнал (2:3 / A4 crop)
    
- **Уровень реализма**: photoreal / editorial photoreal / hyperreal
    
- **Качество**: “high-end magazine editorial, high detail skin texture, fabric microtexture”
    

### B) Цель кадра (это держит “рекламность”)

- **Тип контента**: lingerie ad / glamour editorial / lookbook / teasing lifestyle
    
- **Что продаём**: set / bra / corset / heels / boots
    
- **Где будет копирайт**: “negative space for headline / CTA space”
    

### C) Art-direction preset (пресет стиля)

Это “главный якорь”, который должен включаться первым:

- “Playboy-style glamour editorial”
    
- “Luxury boudoir campaign”
    
- “Neon night glamour”
    
- “Vintage pin-up glamour”
    
- “High-fashion lingerie runway editorial”  
    (ниже дам готовые пресеты)
    

### D) Модель (adult 25+ как обязательный якорь)

- **Возраст**: _adult woman, 25+_
    
- **Телосложение**: slim / athletic / curvy / toned
    
- **Рост/пропорции**: long legs / defined waist / elegant posture
    
- **Лицо**: soft features / sharp jawline / full lips / high cheekbones
    
- **Эмоция/вайб**: confident / playful / provocative (но без explicit)
    
- **Глаза/взгляд**: direct gaze / over-the-shoulder / dreamy
    

### E) Beauty (волосы, мейк, ногти)

- Волосы: length + texture + styling (blowout waves / sleek straight / messy bun / wet look)
    
- Makeup: nude-gloss / smoky eyes / winged liner / red lipstick (как “power accent”)
    
- Nails: nude / black / red (добавляет “дороговизну”)
    

### F) Wardrobe: lingerie / одежда / обувь

**Lingerie**

- Тип: balconette set / bustier / corset / bodysuit / lingerie dress / bikini
    
- Материал: lace / satin / silk / mesh / “latex-look” / “patent-leather accents”
    
- Детали: garter straps (умеренно), high-waist, minimal straps, cut lines
    

**Shoes**

- Вид: stilettos/pumps / ankle boots / thigh-high boots / sandals
    
- Детали: pointed toe / patent leather / suede / thin straps / metal buckles / platform
    

**Outer layer (опционально, очень “журнально”)**

- oversized blazer / silk robe / trench coat slipped off shoulders (именно “slipped”, а не explicit)
    

### G) Accessories & props (аккуратно дозировать)

- Stockings: sheer / lace-top / subtle fishnet (если нужно “edgy”, но не уходить в fetish-эстетику)
    
- Jewelry: minimal gold hoops / layered chain / statement earrings
    
- Props: mirror, chair, perfume bottle, silk sheets, studio cube
    

### H) Сцена / интерьер

Раздели на режимы:

- **Studio clean**: seamless background, cyclorama, product clarity
    
- **Boudoir luxury**: hotel suite, warm lamps, velvet chair, silk bedding
    
- **Night glamour**: neon, bar/cabaret vibe, corridor, reflective surfaces
    
- **Industrial fashion**: concrete, chrome, brutalist minimal
    

### I) Поза + действие (лучше, чем просто “pose”)

- Основа: standing / sitting / reclining / leaning
    
- Body language: elongated neck, relaxed shoulders, arched back _умеренно_, weight shift
    
- Действие: adjusting blazer lapel / holding heel strap / turning mid-step / looking back
    

### J) Камера и композиция

- Shot: full-body / three-quarter / medium / beauty close-up
    
- Angle: eye level / slight low angle (удлиняет ноги)
    
- Lens vibe: 35mm editorial / 50mm portrait / 85mm beauty
    
- Focus: sharp face + crisp lingerie details, soft background bokeh
    
- “Negative space for typography”
    

### K) Свет (делает “Playboy-журнал”)

- Softbox key light + subtle rim light
    
- Beauty dish для лица
    
- Low-key moody (контраст) vs high-key clean (каталог)
    
- Highlight control: glossy highlights on satin/latex-look, no blown whites
    

### L) Цвет, грейдинг, ретушь

- Grading: warm-neutral editorial / cool chrome / filmic
    
- Retouch: fashion retouch, natural skin texture, no plastic skin
    
- Detail: fabric microtexture, realistic seams, clean edges
    

### M) Ограничения / negatives (обязательный модуль)

- adult woman 25+, tasteful glamour lingerie
    
- no explicit nudity / no visible genitals / no sexual act
    
- no minors
    
- no text, no watermark, no logos
    
- no extra fingers/limbs, no warped hands, no distorted face

## 2) Ползунок Boldness (чтобы “на грани”, но управляемо)

Сделай 4 уровня и просто меняй несколько фраз:

**B0 — Lingerie ad safe**: “tasteful lingerie campaign, covered nipples, elegant pose”  
**B1 — Glamour editorial**: “provocative glamour editorial, implied nudity, strategic coverage”  
**B2 — Playboy-like glamour**: “bold glamour, topless implied but not explicit, tasteful framing”  
**B3 — On the edge (но всё ещё non-explicit)**: “very bold glamour, teasing pose, cinematic low-key lighting, no explicit nudity, no sexual act”

Для соцсетей почти всегда пригодится B0–B1; для “журнального” — B1–B3.
